Steps to compile and execute the Poker program:

1. The test inputs are read from a file named input.txt
2. There are three sections in the input.txt file separated by blank line.
   First section - Find Category of the hand
   Second section - Find best permutation
   Third section - Find winner
   (On Linux)
3. g++ -std=c++11 poker.cpp hand.cpp hand.hpp -o poker
4. ./poker
5. The results are displayed on the console.
